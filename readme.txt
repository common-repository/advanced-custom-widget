Custom any widget for wordpress

== Description ==

Advanced Custom Widget is a plugin that allows flexible widget customization without touching a single line of code.

**Features**

Customize text color widgets.
Customize text widget location.
Customize the widget background color.
Customize widget wallpapers.
Custom padding widget.
Custom margin widget.
Add custom class.

[Plugin Homepage](http://longvietweb.com/plugins/advanced-custom-widget)

== Installation ==

1. Upload the entire 'custom-widget' folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
